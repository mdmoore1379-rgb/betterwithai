# Client Onboarding Checklist

**For every new client (especially after Roadmap purchase or project kickoff)**

## Immediate (within 24h of payment)

- [ ] Send Welcome Email (use template in /playbook/templates)
- [ ] Add to CRM / client tracker (simple Google Sheet or future Supabase)
- [ ] Book Kickoff Call (use CalendarAgent or Calendly specific event)
- [ ] Share Intake Form (Google Form or Tally)
- [ ] Grant access to shared folder (Drive/Notion)
- [ ] Send signed MSA + SOW confirmation + payment receipt

## Before Kickoff Call

- [ ] Review intake responses
- [ ] Prepare 3-5 key questions
- [ ] Pull relevant past examples or benchmarks
- [ ] Confirm payment status (AccountingAgent)
- [ ] Block 15 min prep time

## During/After Kickoff

- [ ] Record key decisions and next milestones
- [ ] Set up project board / Trello / Notion page
- [ ] Schedule first deliverable review
- [ ] Confirm communication preferences (Slack, email, weekly calls?)
- [ ] Send summary email with action items and timeline

## 30-Day Check

- [ ] Quick pulse survey or call
- [ ] Confirm value delivered so far
- [ ] Discuss upsell / next project / retainer if appropriate
- [ ] Update internal notes

**Owner Note:** The Ops Leader + OnboardingAgent can drive most of this once connected to email/calendar tools.
