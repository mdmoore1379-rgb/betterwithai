# Better With AI — Business Owner Playbook

This is the complete operating system for running betterwithai as a full-time, high-leverage AI consulting business.

**Goal:** Everything you (Michael) need to produce excellent work, close deals with minimal sales effort, deliver on time, get paid cleanly, and stay sane.

## Core Principles

- Take the work seriously. Don't take ourselves too seriously.
- Self-serve where possible (especially AI Roadmap).
- Clear agreements + easy payments upfront.
- Use agents to multiply output (see /agents).
- One clear next action per client touch.
